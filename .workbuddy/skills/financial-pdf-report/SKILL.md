# 金融 PDF 报告生成器

## 概述

基于 fpdf2 库生成符合中文排版规范的金融分析 PDF 报告。报告自动遵循标准格式：
宋体（SimSun）、五号字（10.5pt）、1.5 倍行距、0 段间距、两端对齐。

## 排版规范（硬性要求）

| 规范项 | 参数值 |
|--------|--------|
| 正文字体 | 宋体（SimSun），从 `C:/Windows/Fonts/simsun.ttc` 加载 |
| 正文字号 | 五号 = 10.5pt |
| 行距 | 1.5 倍 = 15.75pt ≈ 5.56mm |
| 段间距 | 0（段落之间无额外间距） |
| 对齐方式 | `multi_cell(align='J')` 两端对齐 |
| 页面 | A4 竖版 (210×297mm) |
| 页边距 | 默认 10mm（上下各 20mm） |

## 使用流程

### 1. 获取数据
- 通过 Tushare MCP 工具获取股票日线数据（`mcp__tushareMcp__daily`）
- 获取基本面数据（`mcp__tushareMcp__daily_basic`：PE/PB/市值等）
- 获取公司信息（`mcp__tushareMcp__stock_company`：上市公司基本信息）

### 2. 计算技术指标
从日线 CSV 数据中计算以下指标（使用 pandas）：
- MA5 / MA10 / MA20 / MA60 — 移动均线
- MACD (DIF, DEA, 柱状图) — 指数平滑异同移动平均线
- RSI(14) — 相对强弱指标
- 布林带 (Bollinger Bands) — 20日均线 ± 2倍标准差

### 3. 生成报告

拷贝 `scripts/generate_report.py` 到工作目录（作为骨架模板），根据需求修改 `build_pdf()` 函数中的内容。

核心 API 由脚本中的 `HTMLToPDF` 类提供：

```python
pdf = HTMLToPDF()

# 封面
pdf.write_cover()

# 章节标题
pdf.write_section_title("一、章节名称")

# 小节标题
pdf.write_sub_title("1.1 小节")

# 正文段落（自动两端对齐）
pdf.write_body("正文内容...")

# 项目符号列表
pdf.write_bullet("列表项文字", "加粗前缀：")

# 数据表格
pdf.write_table(
    ['列1', '列2', '列3'],
    [['行1', '数据', '说明'], ['行2', '数据', '说明']],
    [30, 40, 90]  # 可选：列宽
)

# 免责声明 + AI 标识（固定放在文末）
pdf.write_disclaimer()
pdf.write_ai_tag()

# 输出
pdf.output("报告文件名.pdf")
```

**重要**：生成报告后，务必检查 `.workbuddy/` 目录不在 git 待提交列表中。

### 4. 报告结构模板

推荐的四部分结构：
1. **"线"分析** — 概念解释（K线/曲线含义）、价格走势回顾、统计概览（涨跌天数、最大涨跌幅等）、月度/周期收益分析
2. **基本面分析** — 概念解释、公司概况、估值指标（PE/PB/PS/市值）、竞争优势、风险提示
3. **技术面分析** — 概念解释、均线系统（MA排列判断）、MACD金叉/死叉判断、RSI超买超卖、布林带位置、综合信号汇总
4. **综合分析与投资建议** — 三维汇总表、长短期投资建议、核心关注点、风险提示

### 5. 图表规范（如有统计图表）

每个嵌入或引用的统计图表必须满足：
- **标号**：如图 1、图 2、表 1、表 2，按出现顺序编号
- **标题**：简洁描述图表内容，如"图 1：贵州茅台过去一年收盘价走势"
- **解读**：紧跟图表之后的段落需要对图表内容进行解读，说明趋势、异常点和结论

示例：
```
表 1：近12个月月度涨跌幅
[表格数据]

从表 1 可以看出，贵州茅台在过去12个月中有 7 个月录得下跌，其中...
```

## 环境要求

- Python 3.8+
- **必须** `fpdf2`（`pip install fpdf2`）
- 可选 `pandas`（数据处理）
- Windows 系统自带宋体（`C:/Windows/Fonts/simsun.ttc`）
