# 北大BA工作坊 — 量化交易公益课

本项目是北京大学商业分析（BA）工作坊量化交易公益课程的实践项目，使用 Tushare 金融数据接口获取 A 股市场数据，通过 Python 进行数据分析和可视化。

## 项目内容

### 数据获取
- 通过 Tushare Pro API 获取沪深 A 股日线交易数据
- 覆盖字段：开盘价、最高价、最低价、收盘价、涨跌幅、成交量、成交额

### 涉及股票
| 股票代码 | 股票名称 | 数据区间 |
|---------|---------|---------|
| 600519.SH | 贵州茅台 | 2025-07-04 ~ 2026-07-03 |
| 688256.SH | 寒武纪 | 2026年6月 |

### 可视化
- **静态图表**（matplotlib）：收盘价曲线、K线图
- **交互式图表**（ECharts）：支持缩放、拖拽、悬停查看详情的动态曲线图

## 文件说明

| 文件 | 说明 |
|------|------|
| `贵州茅台_600519_日线数据.csv` | 贵州茅台过去一年日线交易数据 |
| `贵州茅台_600519_收盘价曲线.png` | matplotlib 生成的静态收盘价曲线图 |
| `贵州茅台_600519_收盘价曲线_交互版.html` | ECharts 交互式平滑曲线图（支持缩放） |
| `寒武纪_688256_K线图_2026年6月.html` | 寒武纪2026年6月交互式K线图 |
| `plot_maotai.py` | 数据读取 + matplotlib 绘图 + CSV 导出脚本 |
| `build_interactive_chart.py` | ECharts 交互式图表生成脚本 |
| `maotai_data.json` | Tushare API 原始返回数据（JSON） |

## 环境依赖

- Python 3.8+
- pandas
- matplotlib
- Tushare Pro（数据接口）

## 使用方法

```bash
# 生成静态曲线图
python plot_maotai.py

# 生成交互式图表
python build_interactive_chart.py
```

交互式 HTML 图表可直接在浏览器中打开，支持：
- 🖱 鼠标滚轮缩放
- 📊 底部滑块选择时间区间
- 🔍 悬停查看当日完整OHLCV数据
