## 第一部分：复利 — 世界第八大奇迹

### 1.1 什么是复利？

**复利**就是"利滚利"——利息也能产生利息。

举个最简单的例子：

> 你有 100 元，年利率 10%。
>
> - **单利**：每年只拿原始 100 元的 10%，即每年固定得 10 元利息。
> - **复利**：第一年得 10 元利息，变成 110 元；第二年按 110 元算，得 11 元利息，变成 121 元……

第一年两种方式没区别，但时间一长，差距是指数级的。

### 1.2 复利公式

```
终值 = 本金 × (1 + 年化收益率)^年数
F = P × (1 + r)^n
```

| 符号 | 含义 | 举例 |
|------|------|------|
| F | 最终金额 | 10年后有多少钱 |
| P | 初始本金 | 投入 10 万元 |
| r | 每期收益率 | 年化 10%，即 0.10 |
| n | 期数（年数） | 投资 10 年 |

### 1.3 一个震撼的数字对比

假设初始本金 **10 万元**，年化收益率 **10%**：

| 年限 | 单利 | 复利 |
|------|------|------|
| 1 年 | 11.0 万 | 11.0 万 |
| 5 年 | 15.0 万 | 16.1 万 |
| 10 年 | 20.0 万 | **25.9 万** |
| 20 年 | 30.0 万 | **67.3 万** |
| 30 年 | 40.0 万 | **174.5 万** |
| 40 年 | 50.0 万 | **452.6 万** |

20 年时复利已是单利的 **2.2 倍**，40 年时是 **9 倍**。复利的核心秘密不是收益率有多高，而是**时间足够长**。

### 1.4 72 法则 — 快速心算

想知道本金翻倍需要多少年？

```
翻倍年数 ≈ 72 ÷ 年化收益率(%)
```

| 年化收益率 | 翻倍约需年数 |
|-----------|-------------|
| 3% | 24 年 |
| 5% | 14.4 年 |
| 8% | 9 年 |
| 10% | 7.2 年 |
| 15% | 4.8 年 |
| 20% | 3.6 年 |

> 同样翻一倍，年化 5% 要等 14 年，年化 15% 不到 5 年。这就是为什么投资中"提高几个点的收益率"意义重大。

### 1.5 复利在投资中的实际应用

**场景一：每月定投**

每月投入 5000 元，年化收益率 8%，坚持 20 年：

- 总投入：5000 × 12 × 20 = 120 万元
- 最终市值：约 **294 万元**

**场景二：股息再投资**

持有一只每年分红 3% 的股票，把分红再买入更多股票，相当于把分红叠加到了复利中。10 年后分红再投的持仓比不投的多出约 34%。

> 核心启示：**越早开始、越稳定执行、越长久坚持，复利的效果越惊人。**

## 第二部分：年化收益率 — 如何统一衡量收益

### 2.1 简单收益率 vs 年化收益率

| | 简单收益率 | 年化收益率（CAGR） |
|---|---|---|
| **怎么算** | (终值 - 本金) / 本金 | 考虑了时间因素 |
| **能比吗** | 不同期限不能直接比 | 统一折算为"每年"后可比 |

**例子**：

> 你投资 10 万，3 年后变成 15 万。
>
> - 简单收益率：50%
> - 年化收益率：不太好直接看出来

### 2.2 CAGR 公式

**CAGR**（Compound Annual Growth Rate，复合年化增长率）：

```
CAGR = (终值 / 本金)^(1/年数) - 1
```

上面的例子：

```
CAGR = (15 / 10)^(1/3) - 1
     = 1.5^0.333 - 1
     ≈ 14.47%
```

意味着这笔投资如果每年稳定增长 14.47%，3 年后 10 万会变成 15 万。

### 2.3 为什么年化很重要？

| 产品 | 收益描述 | 年化后可比吗 |
|------|---------|-------------|
| 银行理财 A | 91 天，收益 1,200 元（本金 10 万） | 年化 ≈ 4.81% |
| 基金 B | 2 年，收益 11,000 元（本金 10 万） | 年化 ≈ 5.37% |
| 股票 C | 5 个月，赚了 8% | 年化 ≈ 19.6% |

统一成年化后，才能公平比较。

### 2.4 年化收益率的常见误导

**误导 1："去年翻倍"**

去年翻倍（+100%），今年亏了 50%，两年总收益是多少？

很多人以为是 +50%。实际上：

```
第一年：10 万 → 20 万
第二年：20 万 → 10 万
两年总收益：0%
年化 CAGR：0%
```

**误导 2：把短期高收益简单乘以时间**

某策略一个月赚了 10%，不等于年化 120%。策略可能在特定行情下有效，不能线性外推。

## 第三部分：股票是什么

### 3.1 股票的本质

**股票**是股份公司发行的所有权凭证。买入一家公司的股票，就相当于买下了这家公司的一小部分。

- 你持有 100 股贵州茅台 = 你是茅台的微型股东
- 公司赚钱了 → 分红给你，或者股价上涨
- 公司出问题了 → 股价下跌，你亏钱

### 3.2 赚钱的两种方式

| 方式 | 怎么赚 | 举例 |
|------|--------|------|
| **资本利得** | 低价买、高价卖 | 100 元买入，150 元卖出，赚 50 元 |
| **股息分红** | 公司把利润分给股东 | 持有 1000 股，每股分红 2 元，到手 2000 元 |

### 3.3 A 股主要板块

中国内地股市（A 股）主要分为：

| 交易所 | 板块 | 代码特征 | 特点 |
|--------|------|---------|------|
| 上交所 | 主板 | 600xxx, 601xxx | 大盘蓝筹，规模大 |
| 上交所 | 科创板 | 688xxx | 科技企业，允许未盈利上市 |
| 深交所 | 主板 | 000xxx, 001xxx | 中等规模公司 |
| 深交所 | 创业板 | 300xxx | 成长型企业，波动大 |
| 北交所 | 北交所 | 8xxxxx | 专精特新中小企业 |

### 3.4 股票代码怎么看

```
688256.SH → 寒武纪
 ↑↑↑    ↑↑
 代码   上交所(SH) / 深交所(SZ) / 北交所(BJ)
```

Tushare 中查询时要带后缀，比如 `688256.SH`。

## 第四部分：看懂 A 股盘面

### 4.1 核心盘面术语

| 术语 | 含义 | 举例 |
|------|------|------|
| **开盘价** | 每个交易日第一笔成交价 | 今天 9:30 开盘价 100 元 |
| **收盘价** | 最后一笔成交价 | 今天 15:00 收盘价 105 元 |
| **最高价** | 当天最高成交价 | 盘中最高冲到 108 元 |
| **最低价** | 当天最低成交价 | 盘中最低砸到 98 元 |
| **昨收** | 上一个交易日的收盘价 | 昨天收盘 99 元 |
| **涨跌幅** | (今收 - 昨收) / 昨收 × 100% | (105-99)/99 = +6.06% |
| **涨停** | A 股主板 ±10%，科创/创业 ±20% | 涨到上限就"封板"了 |
| **跌停** | 同上，向下跌到下限 | 跌停就卖不掉了 |
| **成交量** | 当天成交的股票数量（手） | 1 手 = 100 股 |
| **成交额** | 当天成交的总金额 | 100 万股 × 100 元 = 1 亿 |
| **换手率** | 成交量 / 流通股本 | 5% 说明当天 5% 的股票换了主人 |

### 4.2 中国的红涨绿跌

A 股惯例（和国际相反）：

| 颜色 | 含义 | 出现场景 |
|------|------|---------|
| 🔴 **红色** | 上涨 / 阳线 | 收盘价 > 开盘价 |
| 🟢 **绿色** | 下跌 / 阴线 | 收盘价 < 开盘价 |

### 4.3 A 股交易规则速览

| 项目 | 规则 |
|------|------|
| 交易时间 | 9:30-11:30, 13:00-15:00（周一至周五） |
| 交易单位 | 1 手 = 100 股，买入必须是 100 股的整数倍 |
| T+1 制度 | 今天买入，明天才能卖出（T+1） |
| 涨跌停限制 | 主板 ±10%，科创/创业板 ±20%，北交所 ±30% |
| 最小变动 | 0.01 元 |
| 印花税 | 卖出时收取 0.05%（单边征收） |

### 4.4 重要指数

| 指数 | 含义 |
|------|------|
| 上证指数（000001.SH） | 上交所所有股票的综合指数，"大盘"通常指它 |
| 深证成指（399001.SZ） | 深交所 500 只代表性股票 |
| 沪深 300（000300.SH） | 沪深两市最大的 300 只股票 |
| 中证 500（000905.SH） | 剔除沪深 300 后的 500 只中盘股 |
| 创业板指（399006.SZ） | 创业板 100 只代表股票 |
| 科创 50（000688.SH） | 科创板 50 只龙头 |

## 第五部分：K 线图入门

### 5.1 一根 K 线长什么样

```
        最高价 ———————————————— 上影线顶端
          |
          |  ← 上影线
          |
  ——————— 实体顶部（阳线=收盘价；阴线=开盘价）
  |       |
  | 实体  |  ← 阳线红色（涨）/ 阴线绿色（跌）
  |       |
  ——————— 实体底部（阳线=开盘价；阴线=收盘价）
          |
          |  ← 下影线
          |
        最低价 ———————————————— 下影线底端
```

### 5.2 阳线和阴线

| 阳线（红色） | 阴线（绿色） |
|-------------|-------------|
| 收盘 > 开盘 | 收盘 < 开盘 |
| 当天涨了 | 当天跌了 |
| 实体越大 = 涨势越猛 | 实体越大 = 跌势越猛 |

### 5.3 影线的含义

- **长上影线**：盘中冲高但被砸回来了，上方抛压重
- **长下影线**：盘中急跌但被拉回来了，下方有支撑
- **十字星**（几乎没有实体）：多空在当前位置分歧很大，可能是变盘信号

### 5.4 常见 K 线组合形态

| 形态 | 特征 | 信号 |
|------|------|------|
| **三连阳** | 连续三根实体阳线 | 强烈看涨 |
| **三连阴** | 连续三根实体阴线 | 强烈看跌 |
| **锤子线** | 长下影 + 小实体（阳），处于下跌趋势末尾 | 可能反转向上 |
| **吊颈线** | 长下影 + 小实体（阴），处于上涨趋势末尾 | 可能反转向下 |
| **早晨之星** | 阴线 → 十字星 → 阳线 | 底部反转信号 |
| **黄昏之星** | 阳线 → 十字星 → 阴线 | 顶部反转信号 |

## 第六部分：常用技术指标

### 6.1 移动平均线（MA）

**概念**：最近 N 天的收盘价平均值，连成一条线。

```
MA(N) = 最近 N 天收盘价之和 / N
```

| 均线 | 天数 | 代表 |
|------|------|------|
| MA5 | 5 日 | 超短期趋势（一周） |
| MA10 | 10 日 | 短期趋势（两周） |
| MA20 | 20 日 | 中期趋势（约一个月） |
| MA60 | 60 日 | 中长期趋势（一季度） |
| MA120 | 120 日 | 长期趋势（半年线） |
| MA250 | 250 日 | 年线（牛熊分界线） |

**金叉**：短期 MA 上穿长期 MA → 看涨信号
**死叉**：短期 MA 下穿长期 MA → 看跌信号

### 6.2 MACD（指数平滑异同移动平均线）

MACD 是均线的高级变体，由三条线组成：

| 组成 | 含义 |
|------|------|
| **DIF（快线）** | 12 日 EMA - 26 日 EMA，反映短期趋势 |
| **DEA（慢线）** | DIF 的 9 日 EMA，反映长期趋势 |
| **柱状图（BAR）** | DIF - DEA 的差值，红柱表示多头、绿柱表示空头 |

**MACD 金叉**：DIF 上穿 DEA → 买入信号
**MACD 死叉**：DIF 下穿 DEA → 卖出信号

**顶背离**：股价新高，但 MACD 没有新高 → 上涨乏力，可能见顶。
**底背离**：股价新低，但 MACD 没有新低 → 下跌动能衰竭，可能见底。

### 6.3 RSI（相对强弱指标）

衡量近期涨跌力度的对比：

```
RSI = 100 - 100 / (1 + RS)
RS = 最近 N 天平均涨幅 / 最近 N 天平均跌幅
```

| RSI 数值 | 含义 |
|----------|------|
| 80 以上 | 超买区，短期涨太多，可能回调 |
| 20 以下 | 超卖区，短期跌太多，可能反弹 |
| 50 附近 | 多空力量均衡 |

### 6.4 成交量（Volume）

**量价关系基本原则**：

| 量 | 价 | 含义 |
|----|-----|------|
| 量增 | 价涨 | 上涨有资金支撑，健康 |
| 量缩 | 价涨 | 上涨乏力，可能有隐忧 |
| 量增 | 价跌 | 抛压重，下跌趋势确认 |
| 量缩 | 价跌 | 卖盘枯竭，可能止跌 |

**天量天价**：成交量达到了极高水平，往往预示着阶段性顶部或底部即将出现。

## 第七部分：量化交易概览

### 7.1 什么是量化交易？

**量化交易**就是用数学模型和计算机程序来做交易决策，而不是凭感觉。

| 传统交易 | 量化交易 |
|---------|---------|
| 凭经验、盘感、消息 | 凭数据、统计、回测 |
| 容易受情绪影响 | 严格执行规则 |
| 一个人能盯的股票有限 | 可以同时监控全市场 |
| 决策不稳定 | 决策一致、可复现 |

### 7.2 量化交易的基本流程

```
1. 策略构思  →  2. 数据获取  →  3. 回测验证
                                     ↓
6. 实盘运行  ←  5. 风险控制  ←  4. 策略优化
```

**1. 策略构思**：提出交易逻辑，比如"当 MA5 上穿 MA10 且放量时买入，当 MA5 下穿 MA10 时卖出"。

**2. 数据获取**：通过 Tushare 等数据源获取历史行情、财务数据等。

**3. 回测验证**：用历史数据模拟交易，看策略在过去的表现如何。

**4. 策略优化**：根据回测结果调整参数，但要警惕**过拟合**——参数调得太精准，只在历史数据上表现好，对未来无效。

**5. 风险控制**：设定止损线、仓位上限、最大回撤容忍度。

**6. 实盘运行**：小资金试跑，观察真实市场中的表现。

### 7.3 关键概念

#### Alpha（阿尔法）和 Beta（贝塔）

| 概念 | 含义 | 通俗理解 |
|------|------|---------|
| **Beta** | 策略的系统性风险，跟随市场波动的程度 | 大盘涨 1%，股票涨 1.5%= Beta 1.5 |
| **Alpha** | 策略的超额收益，跑赢市场的能力 | 大盘涨 10%，你赚 15%，Alpha = +5% |

量化交易的核心追求就是**稳定的正 Alpha**。

#### 夏普比率（Sharpe Ratio）

衡量"每承担 1 单位风险能获得多少超额收益"：

```
夏普比率 = (策略年化收益 - 无风险利率) / 策略年化波动率
```

| 夏普比率 | 评价 |
|----------|------|
| < 1 | 一般，收益勉强覆盖风险 |
| 1 - 2 | 不错 |
| > 2 | 优秀 |
| > 3 | 顶级（极少有策略能做到） |

#### 最大回撤（Max Drawdown）

从最高点到最低点的最大亏损幅度。

> 你有 100 万，先涨到 150 万，然后跌到 90 万，再从 90 万涨到 120 万。
>
> - 最大回撤：(150 - 90) / 150 = **40%**
>
> 即便最终赚了 20%，过程中你也扛过了 40% 的亏损，这个策略的风险很高。

### 7.4 常见量化策略类型

| 策略类型 | 核心逻辑 | 举例 |
|---------|---------|------|
| **趋势跟踪** | 跟着趋势走，顺势而为 | 突破 20 日高点买入，跌破 20 日低点卖出 |
| **均值回归** | 价格会回到均值附近 | 跌多了会反弹，涨多了会回调 |
| **动量策略** | 强者恒强 | 买最近涨得最多的股票 |
| **反转策略** | 物极必反 | 买最近跌得最多的股票（博反弹） |
| **套利策略** | 利用价差 | 同一股票 A+H 两地上市价格不一致 |
| **多因子模型** | 综合多个维度打分 | 同时考虑估值、成长、质量、动量等因素选股 |

### 7.5 常用因子

量化选股中，用来评价一只股票"好不好"的维度叫**因子**：

| 因子类别 | 具体指标 | 含义 |
|---------|---------|------|
| **估值因子** | PE（市盈率）、PB（市净率） | 股票贵不贵 |
| **成长因子** | 营收增长率、利润增长率 | 公司增长快不快 |
| **质量因子** | ROE（净资产收益率）、毛利率 | 公司赚钱能力强不强 |
| **动量因子** | 过去 N 个月涨跌幅 | 最近表现好不好 |
| **波动率因子** | 日收益率标准差 | 股价稳不稳 |
| **情绪因子** | 换手率、融资融券数据 | 市场情绪热不热 |
| **技术因子** | 金叉/死叉、RSI、MACD 等 | 技术面信号 |

## 第八部分：重要公式速查

| 公式名称 | 公式 | 用途 |
|---------|------|------|
| 复利终值 | F = P × (1 + r)^n | 计算本金在复利下的最终金额 |
| 年化收益率 (CAGR) | CAGR = (F/P)^(1/n) - 1 | 统一比较不同期限的收益率 |
| 72 法则 | n ≈ 72 / r(%) | 快速估算本金翻倍所需年数 |
| 简单收益率 | R = (F - P) / P | 不含时间因素的收益率 |
| 移动平均 MA | MA(N) = Σ(最近N天收盘价) / N | 判断趋势方向 |
| RSI | RSI = 100 - 100/(1+RS) | 衡量超买超卖 |
| 夏普比率 | S = (Rp - Rf) / σp | 衡量风险调整后的收益 |
| 最大回撤 | MDD = (峰值 - 谷底) / 峰值 | 衡量策略的最大潜在亏损 |
| 胜率 | W = 盈利次数 / 总交易次数 | 多少比例的交易是赚钱的 |
| 盈亏比 | R = 平均盈利 / 平均亏损 | 赚一次能覆盖亏几次 |

## 第九部分：常见误区与学习建议

### 9.1 新手常见误区

| 误区 | 真相 |
|------|------|
| "跟着消息操作就能赚钱" | 等你看到消息，机构早就布局完了 |
| "技术指标 100% 准确" | 任何指标都有失败率，需要组合验证 |
| "分散投资就是多买几只股票" | 买 10 只同行业股票并不是真正的分散 |
| "止损就是认输，死扛总能回本" | 有些股票永远回不到高点（看看退市股） |
| "回测收益好=实盘一定赚钱" | 回测有生存偏差、未来函数、过拟合等陷阱 |
| "追涨停板稳赚" | 涨停板打开后暴跌的例子比比皆是 |

### 9.2 学习路径建议

```
第一阶段：基础知识
  → 看懂 K 线图、理解基本术语、会计算收益率

第二阶段：工具使用
  → 学会用 Tushare 获取数据、用 Python 处理数据

第三阶段：策略认知
  → 了解主流策略类型、理解回测的意义和局限

第四阶段：动手实践
  → 写一个简单策略、回测、分析结果、迭代优化

第五阶段：实盘探索
  → 小资金验证、严格执行风控、持续学习进化
```
